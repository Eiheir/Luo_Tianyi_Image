# 图片命名规则

## 现有命名规则分析

当前目录下的图片主要采用以下命名格式：
1. 三位数数字编号：`001.jpg`、`002.jpg`、`003.png`、...、`016.jpg`用于（emoticon文件夹内）
2. 四位数数字编号：`0001.png`、`0002.jpg`、`0003.png`、...用于（wallpapers&illustration文件夹内）

## 新命名规则

基于现有规则，制定以下统一的图片命名规则：

### 1. 基本格式

```
[序号].[扩展名]
```

### 2. 各部分说明
#### 序号
- 使用4位数编号，不足4位前面补0
- 范围：`0001` - `9999`
- 按编号顺序递增

#### 扩展名
- 使用小写字母
- 支持：`jpg`、`png`、`jpeg`、`gif`、`bmp`


## 4. 特殊情况处理

### 4.1 重复序号处理
- 检查现有文件，确保新文件名唯一
- 如遇重复，自动递增序号

## 5. 管理建议

1. **定期整理**：每月整理一次图片，统一命名格式
2. **分类存储**：按主题或日期创建子文件夹
3. **备份策略**：定期备份重要图片
4. **命名工具**：使用批量重命名工具提高效率

---

通过遵循以上命名规则，可以有效管理图片文件，提高查找和使用效率，同时保持文件命名的规范性和一致性。
